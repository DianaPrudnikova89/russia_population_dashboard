# Дашборд «Население регионов России: динамика, плотность и контрасты»

**Пет-проект.** Исследование демографических трендов России за 2010–2024 годы с построением интерактивного дашборда.

## 📊 Ссылка на дашборд

🔗 [Открыть интерактивный дашборд в Yandex DataLens] (https://datalens.yandex/w3elo8lbwq4cg)

## 📁 Источники данных

- **Численность населения (среднегодовая, 2010–2024):** Росстат — [fedstat.ru/indicator/31556](https://fedstat.ru/indicator/31556)
- **Площадь субъектов РФ:** Википедия — [ссылка](https://ru.wikipedia.org/wiki/Площадь_субъектов_Российской_Федерации)

## 🔧 Методология

- **Среднегодовая численность** — среднее арифметическое между численностью на начало и конец года (методология Росстата)
- **Плотность населения** = численность населения / площадь региона (чел./км²)
- **Обработка данных:** вертикализация через Power Query в Excel (исходные данные были в горизонтальном формате: годы в столбцах)

## 📈 Структура дашборда

### Вкладка 1 — Долгосрочные тренды (2010–2024)
- Фильтры: субъект РФ, федеральный округ
- Индикаторы: изменение численности за 15, 10 и 5 лет (% и абсолютно)
- Графики: динамика численности и плотности населения России
- Рейтинги: топ-5 по росту населения, антитоп-5 по падению

### Вкладка 2 — Аналитика по годам (срез за выбранный год)
- Фильтры: год, субъект РФ, федеральный округ
- Индикаторы: численность населения, плотность, количество регионов и округов
- Топ-7 по численности и топ-7 по плотности
- Детальная таблица по всем регионам

## 📊 Ключевые выводы

1. **Общая динамика:** За 15 лет население выросло на 2,3% (+3,29 млн), но пик был в 2019 году. С 2020 года — устойчивая убыль (–1,57 млн за 5 лет).

2. **Сверхконцентрация:** 5 регионов-лидеров (Москва, Московская область, Крым, Санкт-Петербург, Краснодарский край) дали +6,68 млн прироста. Остальные 80+ регионов потеряли 3,39 млн человек.

3. **Плотность:** С 2019 по 2024 год плотность снизилась с 8,64 до 8,53 (–0,11), полностью обнулив прирост 2015–2019 годов (+0,08).

4. **Контраст топов:** В 2024 году топ-7 по численности и топ-7 по плотности почти не пересекаются (общие только Москва, Московская область и Санкт-Петербург).
